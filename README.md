# FinTrace v0.1 — Engineering Preview (Proprietary)


> **An auditable portfolio accounting and tax-analysis platform for Indian investors that converts broker statements into reconciled holdings, capital-gains working papers, performance analytics, and explainable reports.**

FinTrace is a quantitative analytics application for retail investors. It allows users to track their portfolios, analyze historical performance, compute complex tax liabilities (FIFO, STCG, LTCG), and interact with a quantitative AI Copilot to query their data.

## 🏗️ Architecture & Documentation
FinTrace relies on a strict Chronology-First Ledger that ensures deterministic output across all quantitative calculations.

- [Architecture Overview](docs/architecture.md)
- [Calculation Methodology](docs/calculation-methodology.md)
- [Supported Data Imports](docs/supported-imports.md)

### Import Flow
1. User uploads Zerodha/Groww CSV or PDF.
2. Ingestion engine parses and normalizes the data.
3. Transactions are sequenced chronologically and saved to PostgreSQL.

### Financial Calculation Flow
1. FIFO matching pairs buys and sells (accounting for splits/bonuses).
2. XIRR is calculated across all generated cash flows.
3. Tax liability is bucketed into STCG and LTCG per Indian IT rules.

## 💻 Getting Started (Local Development)

### Local Setup (uv)
1. Clone the repository.
2. Navigate to `backend/` and copy `.env.example` to `.env`. Update variables.
3. Start the backend: `uv run uvicorn api.main:app --reload`
4. Start the frontend: `npm install` and `npm run dev` in the `frontend/` directory.

### Docker Setup
We include a `docker-compose.yml` to spin up the PostgreSQL database and backend seamlessly.
```bash
docker compose up -d
```

### Test Commands
Run the rigorous financial mathematics test suite:
```bash
cd backend
uv sync
uv run pytest tests/
```

### Sample API Call
```bash
curl -X 'GET' \
  'http://localhost:8000/analytics/portfolio/123/tax' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer <token>'
```

## ⚠️ Known Limitations & Disclaimer
- FinTrace currently does not support options/derivatives trading.
- Missing market prices will cause XIRR to fall back to Cost Basis.
- **Disclaimer**: FinTrace is an engineering preview. The tax reports generated by this system are NOT filing-ready and have not been certified by a Chartered Accountant.

## 🔒 Security & Privacy
We process all user data locally on the server. The LangChain AI Copilot runs in isolated ephemeral SQLite contexts to prevent cross-tenant data leakage. See [SECURITY.md](SECURITY.md) for vulnerability reporting.
