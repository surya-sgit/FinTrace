# Contributing to FinTrace

Thank you for your interest in FinTrace!

Currently, FinTrace is a **proprietary, closed-source** project. The source code is hosted publicly for portfolio showcase, transparency, and auditing purposes only.

**We are not accepting outside contributions, pull requests, or feature requests at this time.**

Please note that forking this repository for commercial or derivative use is strictly prohibited by the [LICENSE](LICENSE).

If you are reviewing this code for a potential collaboration, hiring, or security audit, feel free to explore the architecture and methodology documentation in the `docs/` folder!
