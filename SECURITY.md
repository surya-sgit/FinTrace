# Security Policy

## Supported Versions

FinTrace is currently in the **Engineering Preview (v0.1)** stage. Only the latest commit on the `main` branch is actively tested and maintained.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within FinTrace, please do not create a public issue. 
We take security seriously and ask that you report it privately.

Please contact the repository owner directly via email or private message.
Do not disclose the vulnerability publicly until it has been patched and you have received confirmation from the author.

Thank you for helping keep the platform secure!
